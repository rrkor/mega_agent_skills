# ALGORITHM: Q&A ПО ДОКУМЕНТАМ

## WHEN TO USE
Использовать, если пользователь задаёт вопрос по содержимому файла или текста:
- "что написано про X"
- "найди в файле Y"
- "какие проблемы упомянуты"
- "о чём говорится про Z"

---

## INPUT TYPES

file / user_message: документ лежит в session_documents.
Нет документов (session_documents пустой) → tool='need_clarification', попроси приложить файл или вставить текст.

---

## MAIN FLOW

### STEP 1: Поиск по документу

Действия:
- вызови tool='search_info_from_kb'

args:
    kb_id: session_kb_id
    query: вопрос пользователя
    top_k: 5
    search_type: "hybrid"

Если нужен поиск по конкретному документу:
    filter: {"documentId": document_id}

returns:
    results: список релевантных чанков с текстом


### STEP 2: Финал

Сформируй ответ на основе найденных чанков. Отвечай только на основе найденной информации.
Если ничего не найдено — сообщи пользователю что информация не найдена в документе.
